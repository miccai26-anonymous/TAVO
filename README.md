# Anonymous MICCAI 2026 Submission Code

This repository contains the core code for anonymous review.

## Structure
- eval/
- models/
- scripts/

## Quick Start
1. Prepare environment (Python + required packages)
2. Update dataset paths in scripts
3. Run training/evaluation scripts from scripts/

## Notes
- No private data is included.
- Paths and identifiers are anonymized for double-blind review.
